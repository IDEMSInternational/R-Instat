Download R-Instat

If this is your first time downloading R-Instat, we recommend downloading the Complete Installer. The Complete Installer is the same version of R-Instat as the Basic Installer but also includes an up to date version of R and the .NET Framework, which are required to run R-Instat.

If you do not know if you have the latest version of R, download the Complete Installer. The Basic Installer does not include R or the .NET Framework

If this is your first time downloading R-Instat, we strongly recommend following the installation guide to ensure R-Instat installs correctly.

Prerequisite Software

R

R-Instat requires R to be installed on your machine before it can run. A latest version is included in the Complete Installer. If you need to download it, you can download R here.

.NET Framework

R-Instat is a .NET software and requires an up to date version of the .NET to be installed before it can run. Most new Windows version already have this. An up to date version is included in the Complete Installer. If you need to download it, you can download .NET Framework here.

Rtools (optional)

R-Instat requires Rtools to export data to Excel files. If you do not intend to use this feature, you do not need to download Rtools. It is NOT included in the Complete Installer. If you would like to you can download Rtools here.
