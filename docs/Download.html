Prerequisite Software

R: R-Instat requires R 3.4.4 to be installed on your machine before it can run. R 3.4.4 is included in the Complete Installer. If you need to download it, you can download R 3.4.4 here.

.NET Framework: R-Instat is a .NET software and requires an up to date version of the .NET to be installed before it can run. Most new Windows version already have this. The installation process will tell you if your .NET needs to be updated. If you need to download it, you can download .NET Framework here.

Rtools: R-Instat requires Rtools to export data to Excel files. If you do not intend to use this feature, you do not need to download Rtools. If you would like to you can download Rtools here.
