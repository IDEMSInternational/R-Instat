Help

Comprehensive help for each R-Instat menu item can be accessed from within the individual dialogues. Alternatively, the help menu describes R-Instat and gives access to the whole help system. Each dialogue constructs an R command, and most are from a set of R packages that is automatically installed. Information on each package is provided, including links to the packages own documentation. Interesting data sets are provided by many R packages.
